
# 🛠️ CSS - Outros

---

## 1️⃣ Transparência
```css
div {
    opacity: 0.5; /* 0 = invisível, 1 = opaco */
}
```

---

## 2️⃣ Overflow
```css
div {
    overflow: hidden; /* visible, hidden, scroll, auto */
}
```

---

## 3️⃣ Reset de Estilos
```css
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
```
